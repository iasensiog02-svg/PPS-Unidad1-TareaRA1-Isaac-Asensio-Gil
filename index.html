PPS - Unidad 1 - Tarea RA1 - Isaac

Este documento corresponde a la entrega de la Unidad 1 - Tarea RA1 del módulo de PPS.
La tarea consiste en crear un repositorio en GitHub con nombre Unidad1-TareaRA1-Isaac, configurarlo con MkDocs y GitHub Pages.

Elementos de Python

Aquí se incluye el código de python exportado , explicado con jupyter.
El objetivo es mostrar ejemplos de programación en Python y explicar su funcionamiento.

Ejecución y Depuración

Se documentan los problemas encontrados en el código, cómo se solucionaron y se añaden capturas de pantalla del proceso de depuración.

Pruebas

Se presentan las pruebas realizadas en la carpeta test, explicando los casos de prueba, errores encontrados y resultados obtenidos.
Incluye el código de las pruebas y capturas de la ejecución en el IDE.

Ejecución en Sandbox

Explicación del proceso de ejecución del código en un entorno Sandbox, con capturas de pantalla de la instalación, ejecución y pruebas realizadas.

Reflexión sobre seguridad en lenguajes

Comparación de la infraestructura de seguridad de Python con otros lenguajes (Java, C#, C/C++), destacando ventajas y riesgos.
Se añade una reflexión personal sobre lo aprendido en la tarea.
