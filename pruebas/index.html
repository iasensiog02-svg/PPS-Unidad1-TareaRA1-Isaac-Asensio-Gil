Pruebas de Test

Durante el proceso de depuración se detectaron varios errores en el código y en los tests.
A continuación se detallan uno por uno, con su explicación y la solución aplicada.

Fallos y correcciones

  1. Clase Lavadero
  2. Se llamaba al método _hacer_lavado en lugar de hacerLavado.
  3. Corrección: Cambiar la llamada a self.lavadero.hacerLavado(...).
  1. Test 3
  2. Se lanzaba ValueError cuando debía ser RuntimeError.
  3. Corrección: Sustituir la excepción por RuntimeError.
  1. Test 5
  2. Los ingresos esperados eran 6.
  3. El cálculo correcto es 6.20 (5.00 inicial + 1.20 coste de lavado).
  4. Corrección: Cambiar ingresos esperados de 6 a 6.20.
  1. Test 7
  2. Los ingresos esperados eran 7.50.
  3. Con prelavado a mano (1.50) y secado a mano (1.20), el total correcto es 7.70.
  4. Corrección: Cambiar ingresos esperados de 7.50 a 7.70.
  1. Clase Lavadero – método ejecutar_y_obtener_fases
  2. El método estaba fuera de la clase por falta de tabulación.
  3. Corrección: Añadir la indentación correcta para que quede dentro de la clase.
  1. Test 9
  2. fases_esperadas = [0, 1, 3, 4, 5, 6, 0].
  3. Corrección: Cambiar a [0, 1, 3, 4, 5, 7, 0].
  1. Test 10
  2. fases_esperadas = [0, 1, 2, 3, 4, 5, 6, 0].
  3. Corrección: Cambiar a [0, 1, 2, 3, 4, 5, 7, 0].
  1. Test 11
  2. fases_esperadas = [0, 1, 3, 4, 5, 7, 0]`.
  3. Corrección: Cambiar a [0, 1, 3, 4, 5, 6, 0].
  1. Test 12
  2. fases_esperadas = [0, 1, 3, 4, 5, 7, 8, 0].
  3. Corrección: Cambiar a [0, 1, 3, 4, 5, 6, 0].
  1. Test 13
    • fases_esperadas = [0, 1, 2, 3, 4, 5, 7, 0].
    • Corrección: Cambiar a [0, 1, 2, 3, 4, 5, 6, 0].
  1. Test 14
    • fases_esperadas = [0, 1, 2, 3, 4, 5, 7, 8, 0].
    • Corrección: Cambiar a [0, 1, 2, 3, 4, 5, 6, 0].
  1. Test 15
    • Se llamaba a self.lavadero.hacerLavado(False, False, True).
    • No se puede encerar el coche sin haber realizado el secado a mano.
    • Corrección: Cambiar a self.lavadero.hacerLavado(False, True, True).
